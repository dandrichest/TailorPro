# 🧵 TailorPro

**TailorPro** is a Smart Tailoring Shop Management System built using the .NET Blazor framework. It helps tailoring businesses efficiently manage customers, orders, and measurements in a structured and digital way.

---

## 📖 Project Overview

TailorPro is a web-based application designed to simplify and improve the daily operations of tailoring businesses. Many tailoring shops rely on manual record-keeping, which often leads to lost information, poor organization, and inefficiencies. TailorPro solves this problem by providing a centralized system where all business activities can be managed effectively.

The system allows users to store customer details, track clothing orders, manage measurements, and monitor business performance through a dashboard. With secure authentication, all data is protected and accessible only to authorized users.

This application is intended for tailoring shop owners and staff who want a simple, reliable, and digital solution to manage their workflow. The project is both practical and scalable, with potential for future enhancements like notifications and invoicing.

## 🎯 Project Scope

### ✅ What's Included (IN)
- User authentication (Login & Registration)
- Customer management (Create, Read, Update, Delete)
- Order management system
- Measurement tracking for customers
- Dashboard with business insights
- Responsive user interface

---

### ❌ What's Excluded (OUT)
- Online payment processing
- SMS or email notifications
- Mobile app version
- Advanced analytics
- Multi-branch support

> ℹ️ These exclusions help ensure the project remains achievable within the given timeframe.

---

## ⚙️ Core Features

- Users can create an account and log in securely
- Users can add, edit, view, and delete customer records
- Users can create and manage tailoring orders
- Users can update order status (Pending, In Progress, Completed, Delivered)
- Users can store and manage customer body measurements
- Users can view a dashboard with business statistics
- Users can link orders to customers
- Users can search and navigate records easily

---

## 🧠 User Stories

- As a tailor, I want to add new customers so I can keep track of their details.
- As a tailor, I want to record measurements so I can reuse them later.
- As a tailor, I want to track order progress so I know what is pending or completed.
- As a user, I want secure login so my data is protected.
