# 🧵 TailorPro

TailorPro is a Smart Tailoring Shop Management System built using the .NET Blazor framework. It helps tailoring businesses manage customers, orders, and measurements efficiently.

## 🚀 Features
- User Authentication
- Customer Management (CRUD)
- Order Management (CRUD)
- Measurement Tracking
- Dashboard Overview
